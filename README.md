# Basic_Portfolio_Website

# 🌐 Personal Portfolio Website

A simple and responsive personal portfolio website built using **HTML, CSS, and JavaScript**. This project was created to showcase my profile, skills, education, and projects in a clean and modern way.

---

## 📌 About the Project

This is my personal portfolio website where visitors can learn about me, explore my projects, view my technical skills, and contact me.

The website is designed with a modern UI using a glassmorphism theme and works smoothly on both desktop and mobile devices.

---

## ✨ Features

- Responsive design for desktop and mobile
- Modern glassmorphism user interface
- Smooth scrolling navigation
- Mobile-friendly navigation menu
- Hero section with profile image
- About Me section
- Featured Projects section
- Technical Skills section
- Education section
- Contact section with social links

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts
